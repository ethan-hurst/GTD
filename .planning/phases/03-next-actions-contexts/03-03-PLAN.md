**Create `src/lib/components/ActionDetailPanel.svelte`:**

An expanded detail view that slides open below the ActionItem when expanded. Allows editing all action fields.

```typescript
interface ActionDetailPanelProps {
  item: GTDItem;
  onSave: () => void;   // Called after saving changes
  onClose: () => void;  // Called to collapse panel
}
```

Layout (inside a slide transition container):
- Title: Editable input field (pre-filled with item.title)
- Notes: Textarea (pre-filled with item.notes, expandable, 3 rows min)
- Context: Dropdown select populated with all contexts from actionState.contexts. Shows current context selected. Include "No context" option. Changing context saves immediately.
- Project: For now, a simple number input or text field for projectId (placeholder for Phase 4 which will add proper project lookup). Label: "Project ID (optional)". In Phase 4 this becomes a searchable dropdown.
- Created: Read-only display of creation date (formatted)
- Modified: Read-only display of last modified date (formatted)
- Action buttons: "Save" (primary) and "Delete" (red, with confirmation)

Behavior:
- Use local $state for each editable field, initialized from item props
- "Save" button: Call `updateItem(item.id, { title, notes, context, projectId })`. Then call `onSave()` which triggers parent to reload actions.
- "Delete" button: Confirm with `confirm("Delete this action?")`. If yes, call `deleteItem(item.id)`, toast "Action deleted", call onSave().
- Auto-save on context change (dropdown onchange handler): Immediately save the new context, no need to click Save. Show brief toast "Context updated".
- Close button (X) in top-right corner: Calls onClose().

Slide transition: Use `transition:slide={{ duration: 200 }}` consistent with ProcessingFlow pattern.

Styling: Match the ProcessingFlow.svelte styling pattern (bg-gray-50 dark:bg-gray-800/50, p-4, rounded-b-lg). Use the same input/button styling from ProcessingFlow.

Import: updateItem, deleteItem from operations.ts. actionState from stores. slide from svelte/transition. toast from svelte-5-french-toast.
