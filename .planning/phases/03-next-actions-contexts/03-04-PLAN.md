**1. Context sidebar (NACT-01, NACT-05):**
- Verify sidebar shows "Next Actions" section with default contexts: @computer, @office, @phone, @home, @errands
- Verify "All" option is shown and selected by default

- Click a second context - verify multi-select works (both highlighted, combined results shown)
- Click "All" - verify all actions shown again

**2. Create custom context (NACT-02):**
- Click "+" button in context list
- Type "meeting" and press Enter

- Hover over @meeting, verify edit/delete icons appear
- Rename it to "@standup" using the edit icon
- Delete @standup using the delete icon (confirm dialog)

- Pick @computer
- Verify toast shows "Added to Next Actions"
- Navigate to /actions (press 'n')
- Verify "Review quarterly report" appears with @computer badge

**4. Complete an action (NACT-03):**
- Click the completion checkbox on an action
- Verify strikethrough + fade-out animation (~1 second)
- Verify toast appears: "Review quarterly report" completed - Undo
- Click Undo within 5 seconds
- Verify action reappears in the list

**5. Drag to reorder:**

- Drag an action to a new position
- Verify smooth flip animation during drag
- Refresh the page
- Verify the new order persists

**9. Empty states:**
