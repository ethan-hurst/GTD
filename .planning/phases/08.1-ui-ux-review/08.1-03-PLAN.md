Polish the Inbox page (+page.svelte):

1. **Layout consistency:**
   - Change outer wrapper from `max-w-4xl mx-auto p-8` to `max-w-4xl mx-auto p-6` to match other list views (ActionList, ProjectList, WaitingForList all use p-6)

2. **Header refinement:**
   - Keep `text-2xl font-bold` for the h1 (consistent with other views)
   - Change item count span: add `tabular-nums` class for stable number rendering (prevents layout shift when count changes)

3. **Process Inbox button:**
   - Add focus ring: `focus:outline-none focus:ring-2 focus:ring-blue-500/40 focus:ring-offset-2`
   - Change `rounded-md` to `rounded-lg` for the large full-width CTA (hero buttons get rounded-lg per research)
   - Add `active:scale-[0.99]` for a micro press feedback
   - Add `shadow-sm hover:shadow-md` for depth progression on hover
   - Change `transition-colors` to `transition-all duration-150`

4. **Stop Processing button:**
   - Add same focus ring pattern
   - Change to `rounded-lg` to match
   - Add `transition-all duration-150`

5. **Empty state (Inbox Zero) enhancement:**
   - Change icon color from `text-gray-300 dark:text-gray-700` to `text-green-400/60 dark:text-green-500/40` (celebratory -- green signals success)
   - Change heading from `text-xl font-medium text-gray-600 dark:text-gray-400` to `text-xl font-semibold text-gray-700 dark:text-gray-300` (slightly stronger -- it's a positive state)
   - Add a subtle decorative touch: wrap the SVG in a div with `w-20 h-20 mx-auto bg-green-50 dark:bg-green-900/20 rounded-full flex items-center justify-center mb-4` so the icon sits in a soft green circle (signals achievement)
