## Decisions Made
- **Shadow over border:** Header now uses `shadow-[0_1px_3px_-1px_rgba(0,0,0,0.1)]` instead of `border-b` for more professional separation
- **Gray-950 dark mode:** Using gray-950 instead of pure black (#000000) for dark mode background to reduce eye strain
- **Minimal utility set:** Created only essential utilities (.focus-ring, .btn-base, .shadow-soft, .shadow-elevated) - avoiding over-engineering a full design system
- **Explicit text optimization:** Added `text-rendering: optimizeLegibility` and `-webkit-font-smoothing: antialiased` globally for professional typography
- **Consistent spacing:** Reduced main content pb-9 to pb-6, used px-6 consistently in header and status bar
