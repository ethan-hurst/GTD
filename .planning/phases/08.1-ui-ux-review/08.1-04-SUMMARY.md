## Decisions Made

1. **Focus ring opacity at 40%** - Provides clear visual indicator without overwhelming the interface
2. **Ring offset for buttons** - Using ring-offset-2 on buttons creates clear separation from button background color
3. **Ring inset for list items** - Keeps focus rings inside component boundaries to prevent layout shift
4. **Transition duration standardized at 150ms** - Fast enough to feel responsive, slow enough to see the transition
5. **Shadow-sm on inputs, shadow-lg on panels** - Creates clear visual hierarchy between form elements and containers
