1. **Start review:** Navigate to /review (or press 'r'). See "Weekly Review" heading, last review info, and "Start Weekly Review" button. If first time, should show "Never completed" with overdue indicator.

2. **Sidebar link:** Check sidebar shows "Weekly Review" between Someday/Maybe and Settings. Should show red "Overdue" badge if no review completed or >7 days.

3. **Progress tracking:** Click "Start Weekly Review". Verify:
   - All 8 steps visible in left sidebar with step numbers
   - Progress bar at 0%
   - First step (Empty Your Inbox) is highlighted/active
   - Step shows description and inbox item count

4. **Step navigation:** Click through steps using:
   - "Next" button at bottom
   - Direct click on step in sidebar
   - "Back" button
   - Verify progress bar updates as steps are marked complete

5. **Item counts:** Check each step shows relevant counts:
   - Inbox: item count from inbox
   - Actions: next action count
   - Waiting: waiting-for item count
   - Projects: project count with stalled count
   - Someday: someday/maybe item count
   - Calendar steps: informational text (no count — not yet integrated)
   - Creative: informational text (no count)

6. **Step completion:** Mark each step complete. Verify:
   - Green checkmark appears on completed steps
   - Progress bar increases
   - Auto-advances to next incomplete step
   - Percentage text updates

7. **Finish review:** After all 8 steps complete:
   - "Finish Review" button appears
   - Click it — confetti animation fires
   - Toast notification: "Weekly review complete!"
   - Returns to start page
   - Last review shows "Today"
   - Sidebar "Overdue" badge disappears

8. **Keyboard shortcut:** Press 'r' from any page — navigates to /review.
