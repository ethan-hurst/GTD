## Decisions Made

- **Show toast on every app load when persistent:** Provides user confirmation that their data is safe, reinforcing trust in the system (CONTEXT requirement)
- **Browser-specific denial guidance:** Detect browser type (Chrome/Safari/Firefox/Edge) and show targeted advice based on platform-specific persistence granting behaviors
- **Only show quota when persistent:** Quota display is only meaningful when storage won't be evicted, reduces clutter in non-persistent state
- **Four-state StatusBar indicator:** Account for Storage API unavailability (older browsers) with gray dot, distinct from amber (not requested), red (denied), green (granted)
- **Loading spinner on request:** Visual feedback during async persistence request prevents double-clicks and shows system responsiveness
