UI structure:
- Header: "Calendars" with a count badge (e.g., "3 of 5 enabled")
- For each calendar in outlookSyncState.calendars:
  - Color dot (small circle matching the calendar color, or default gray)
  - Calendar name (text)
  - Toggle switch on the right side (enabled/disabled)
  - "Default" badge for the user's default calendar
- Empty state: "No calendars found. Click Sync Now to discover calendars."

Toggle switch implementation:
- Use a Tailwind-styled toggle (bg-blue-600 when on, bg-gray-200 when off)
- On toggle, call `outlookSyncState.toggleCalendar(calendarId, newState)`
- Smooth transition

Style to match the existing Settings page card pattern:
- Rounded card with border/shadow
- Consistent with other Settings sections (Data Export, Device Sync, etc.)
- Each calendar row has py-3 px-4, border-b between rows
- Dark mode support

Mobile responsive:
- Touch-friendly toggle size (min 44px height)
- Text truncation for long calendar names
