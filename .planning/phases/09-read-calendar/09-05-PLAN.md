Create `src/lib/stores/outlook-sync.svelte.ts` following the Svelte 5 $state runes pattern:

```typescript
import { syncCalendarEvents, fetchCalendars } from '$lib/services/graph/calendar';
import { getAllSyncMeta, upsertSyncMeta, clearAllSyncMeta } from '$lib/db/operations';
import { calendarState } from '$lib/stores/calendar.svelte';
import type { SyncMeta } from '$lib/db/schema';

type OutlookSyncStatus = 'idle' | 'syncing' | 'error';

class OutlookSyncStore {
  status = $state<OutlookSyncStatus>('idle');
  lastSyncTime = $state<Date | null>(null);
  error = $state<string | null>(null);
  calendars = $state<SyncMeta[]>([]);
  syncProgress = $state<{ current: number; total: number }>({ current: 0, total: 0 });

  isSyncing = $derived(this.status === 'syncing');
  hasError = $derived(this.status === 'error');

  /**
   * Load saved sync metadata (calendar list + delta links)
   */
  async loadSyncMeta(): Promise<void> {
    this.calendars = await getAllSyncMeta();
  }

  /**
   * Fetch available calendars from Outlook and save to syncMeta
   * Called after first login to discover calendars
   */
  async discoverCalendars(): Promise<void> { ... }

  /**
   * Sync all enabled calendars
   * For each enabled calendar: load delta link from syncMeta, call syncCalendarEvents, save new delta link
   */
  async syncAll(): Promise<void> { ... }

  /**
   * Sync a single calendar by ID
   */
  async syncCalendar(calendarId: string): Promise<void> { ... }

  /**
   * Toggle a calendar's enabled state
   */
  async toggleCalendar(calendarId: string, enabled: boolean): Promise<void> { ... }

  /**
   * Clear all sync data (for disconnect)
   */
  async clearAll(): Promise<void> { ... }
}

export const outlookSyncState = new OutlookSyncStore();
```

Implementation details for key methods:

**discoverCalendars():**
1. Call `fetchCalendars()` from calendar.ts
2. Load existing syncMeta
3. For each Outlook calendar not already in syncMeta, create a new SyncMeta entry with enabled=true for default calendar, enabled=false for others
4. Save all to Dexie via upsertSyncMeta
5. Reload calendars state

**syncAll():**
1. Set status='syncing', error=null
2. Get all enabled calendars from this.calendars
3. Set syncProgress = { current: 0, total: enabledCalendars.length }
4. For each enabled calendar:
   a. Load existing deltaLink from calendar's syncMeta
   b. Call syncCalendarEvents(calendarId, deltaLink)
   c. If success, update syncMeta with new deltaLink and lastSyncAt
   d. Increment syncProgress.current
5. On completion: set status='idle', lastSyncTime=new Date()
6. Call calendarState.loadEvents() to refresh the calendar view
7. On any error: set status='error', error=message, but continue with remaining calendars

**toggleCalendar(calendarId, enabled):**
1. Find the calendar in this.calendars
2. Update enabled state
3. Save via upsertSyncMeta
4. Update calendarState.outlookCalendarsEnabled Set
5. Reload calendars state

**clearAll():**
1. Call clearAllSyncMeta()
2. Reset all state to defaults
3. (Note: clearOutlookEvents is called by auth store on disconnect)

Important: This store is SEPARATE from the device sync store (sync.svelte.ts). They serve different purposes: sync.svelte.ts is for Netlify Blobs device-to-device sync, this is for Outlook calendar sync.

Create `src/lib/components/SyncStatusIndicator.svelte`:

A small status display showing:
- Last sync time (e.g., "Last sync: 2 min ago" or "Never synced")
- Current state indicator (green dot = idle, pulsing = syncing, red dot = error)
- Error message on hover/click if in error state

Use relative time formatting (now, X min ago, X hours ago, yesterday, date).

```svelte
<script lang="ts">
  import { outlookSyncState } from '$lib/stores/outlook-sync.svelte';

  const relativeTime = $derived.by(() => {
    if (!outlookSyncState.lastSyncTime) return 'Never synced';
    const diffMs = Date.now() - outlookSyncState.lastSyncTime.getTime();
    const diffMin = Math.floor(diffMs / 60000);
    if (diffMin < 1) return 'Just now';
    if (diffMin < 60) return `${diffMin}m ago`;
    const diffHours = Math.floor(diffMin / 60);
    if (diffHours < 24) return `${diffHours}h ago`;
    return outlookSyncState.lastSyncTime.toLocaleDateString();
  });
</script>

<div class="flex items-center gap-1.5 text-xs text-gray-500 dark:text-gray-400">
  <!-- Status dot -->
  <span class="..." />
  <span>{relativeTime}</span>
</div>
```

The status dot:
- Green (#10B981) when idle and lastSyncTime exists
- Pulsing blue (#3B82F6) when syncing (use animate-pulse)
- Red (#EF4444) when error
- Gray (#9CA3AF) when never synced

Both components should be compact enough to sit in the calendar toolbar without taking too much space.
