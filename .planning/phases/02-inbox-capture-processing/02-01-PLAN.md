       - Create `formatRelativeTime(date: Date): string` function using `Intl.RelativeTimeFormat`:
         - Instantiate with `new Intl.RelativeTimeFormat('en', { numeric: 'auto', style: 'long' })`
         - Calculate diff in seconds between date and now
         - If abs(diffSecs) < 60: return "just now"
         - If abs(diffMins) < 60: return rtf.format(diffMins, 'minute')
         - If abs(diffHours) < 24: return rtf.format(diffHours, 'hour')
         - If abs(diffDays) < 7: return rtf.format(diffDays, 'day')
         - If abs(diffDays) < 30: return rtf.format(Math.round(diffDays / 7), 'week')
         - Otherwise: return date.toLocaleDateString()
       - Use `Math.round()` for unit calculations to get natural-feeling results
       - Export the function as named export

    3. Create `src/lib/stores/inbox.svelte.ts`:

         - `isProcessing = $state(false)` — whether in sequential processing mode
         - `selectedIds = $state<number[]>([])` — multi-select for bulk actions

         - `selectAll()` — sets selectedIds to all item ids
         - `expandItem(id: number)` — sets expandedId (or null if same id, toggling)
         - `startProcessing()` — sets isProcessing = true, expandedId = first item's id
         - `advanceToNext()` — after processing current item, reload items and expand next item (or end processing if no items left)
