  <action>
    Create `src/lib/components/SearchBar.svelte`:

    **State:**
    - `let query = $state('')` — search input value
    - `let results = $state<GTDItem[]>([])` — search results
    - `let isOpen = $state(false)` — whether dropdown is visible
    - `let selectedIndex = $state(-1)` — keyboard navigation index in results
    - `let searchEl: HTMLInputElement` — ref for programmatic focus

    **Search logic:**
    - Create a debounce helper inline (8-line function, no need for separate file):
      ```typescript
      function debounce<T extends (...args: any[]) => any>(fn: T, delay: number) {
        let timeoutId: ReturnType<typeof setTimeout>;
        return (...args: Parameters<T>) => {
          clearTimeout(timeoutId);
          timeoutId = setTimeout(() => fn(...args), delay);
        };
      }
      ```
    - Create `performSearch` async function: calls `searchItems(query)` from db/search.ts, sets results, sets isOpen = results.length > 0
    - Create `debouncedSearch = debounce(performSearch, 300)`
    - Use `$effect(() => { debouncedSearch(query); })` to trigger search on query change
    - When query is empty, clear results and close dropdown

    **Keyboard navigation:**
    - ArrowDown: increment selectedIndex (wrap to 0)
    - ArrowUp: decrement selectedIndex (wrap to last)
    - Enter: if selectedIndex >= 0, navigate to selected result
    - Escape: close dropdown, clear query, blur input

    **Dropdown results:**
    - Position: absolute, top-full, left-0, right-0, bg-white dark:bg-gray-800, shadow-lg, border, rounded-md, mt-1, max-h-80 overflow-y-auto, z-50
    - Each result item:
      - Title (font-medium text-sm)
      - Type badge: small pill showing item type (inbox, next-action, project, waiting) with color coding:
        - inbox: blue
        - next-action: green
        - project: purple
        - waiting: amber
      - Note preview: first 60 chars of notes, text-xs text-gray-500, truncated
    - Highlighted item (selectedIndex match): bg-gray-100 dark:bg-gray-700
    - Click handler: navigate to appropriate page (for now, all go to '/' since only inbox exists; future phases will add routing)

    **Close on click outside:**
    - Use a wrapper div with a backdrop: when dropdown is open, add an invisible overlay that closes on click

    **Styling:**
    - Input: w-full bg-gray-100 dark:bg-gray-800 border-0 rounded-md px-3 py-1.5 text-sm placeholder-gray-400 focus:ring-2 focus:ring-blue-500 focus:bg-white dark:focus:bg-gray-700
    - Placeholder text: "Search... (Cmd+K)"
    - Search icon (magnifying glass SVG) on left inside input via relative positioning
