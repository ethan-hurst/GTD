## Decisions Made

**1. Use real browser IndexedDB instead of fake-indexeddb**
- **Rationale:** Browser mode provides real IndexedDB API, eliminating need for polyfill. More accurate testing, better compatibility with Dexie.
- **Impact:** setup.ts uses `indexedDB.databases()` and `deleteDatabase()` instead of `IDBFactory` reset.

**2. Install @playwright/test separately from playwright**
- **Rationale:** playwright package provides browser binaries, but @playwright/test is needed for E2E test API (defineConfig, devices).
- **Impact:** Added @playwright/test to devDependencies (blocking dependency).

**3. Use raw svelte() plugin in vitest.config.ts**
- **Rationale:** Plan specified not using sveltekit() plugin in vitest config to avoid interference with browser mode.
- **Impact:** Separate plugin configuration from vite.config.ts.
