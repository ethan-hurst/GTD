## Decisions Made

1. **Chart.js dynamic import for code splitting** - Import Chart.js inside $effect to avoid loading 200KB+ library on every page load. Only loads when user visits /analytics.

2. **Password in memory only** - Password input is not persisted or stored in localStorage. User must re-enter password after page refresh for security (admin-only page).

3. **Standalone admin page** - Dashboard does not use app sidebar layout since it's an admin-only page, not part of normal GTD workflow.

4. **Date range selector** - Three preset ranges (7d/30d/90d) instead of custom date picker for simplicity and common use cases.

## User Setup Required

**ANALYTICS_PASSWORD environment variable required in Netlify:**

1. Go to Netlify dashboard → Site settings → Environment variables
2. Add variable: `ANALYTICS_PASSWORD` = `[your-secure-password]`
3. Deploy site for changes to take effect

**Verification:** Visit `/analytics` and enter the password you configured.
