<task type="auto">
  <name>Task 2: Create Netlify Function for event ingestion with visitor hashing and daily aggregation</name>
  <files>

  <action>
Create `netlify/functions/analytics-utils.ts`:
- `generateVisitorId(request: Request, siteDomain: string): Promise<string>`
  - Extract IP from `x-nf-client-connection-ip` header, fallback to `x-forwarded-for` first entry, fallback to `'unknown'`
  - Extract User-Agent from `user-agent` header, fallback to `'unknown'`
  - Get daily salt from Netlify Blobs `analytics` store at key `salt/${today}` (YYYY-MM-DD format)
  - If no salt exists, generate 32 random bytes via `crypto.getRandomValues(new Uint8Array(32))`, convert to hex, store in Blobs with metadata `{ createdAt: Date.now() }`
  - Hash `salt + siteDomain + ip + userAgent` using `crypto.subtle.digest('SHA-256', ...)` and return hex string
  - CRITICAL: Never persist raw IP or User-Agent -- only the hash
- `getDailyAggregate(store, date: string): Promise<DailyAggregate>`
  - Fetch from `daily/${date}` key, return parsed JSON or default: `{ date, pageviews: {}, events: {}, visitors: [], uniqueVisitors: 0, referrers: {} }`
  - Note: `visitors` is stored as array (Sets don't serialize to JSON -- pitfall from research)
- `updateDailyAggregate(store, date: string, aggregate: DailyAggregate): Promise<void>`
  - JSON.stringify and store at `daily/${date}`
- Define and export `DailyAggregate` interface:
  ```
  { date: string; pageviews: Record<string, number>; events: Record<string, number>; visitors: string[]; uniqueVisitors: number; referrers: Record<string, number> }
  ```

Create `netlify/functions/analytics-track.mts`:
- Accept POST only (return 405 for other methods)
- Parse JSON body: `{ event, url, timestamp, ...properties }`
- Validate: `event` must be string, `url` must be string
- Check DNT header server-side too: if `request.headers.get('dnt') === '1'`, return 200 with `{ tracked: false }` (silent success, don't error)
- Generate visitor ID via `generateVisitorId(request, 'gtd-app')`
- Get today's date as YYYY-MM-DD
- Fetch existing daily aggregate via `getDailyAggregate`
- Update aggregate:
  - Increment `pageviews[url]` if event is 'pageview'
  - Increment `events[event]` for all events
  - Add visitorId to visitors array if not already present (use `includes()` check -- no Set)
  - Update `uniqueVisitors` count from visitors array length
  - Track referrer from properties if present
- Save updated aggregate via `updateDailyAggregate`
- Return 200 with `{ tracked: true }`
- Wrap in try/catch -- return 500 on error, log to console.error

Follow the same patterns as sync-push.mts for Response construction and error handling. Import `getStore` from `@netlify/blobs` and `Context` from `@netlify/functions`.
  </action>
